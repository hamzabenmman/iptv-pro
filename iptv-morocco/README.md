# 🏆 IPTV Pro - Premium IPTV Landing Page

A high-converting landing page for IPTV services targeting the **Arab world** and international audiences. Features **automatic language detection** (80+ languages), **WC2026-inspired gold design**, and **AI-powered SEO optimization**.

## ✨ Features

- 🌍 **80+ Langues** - Auto-detects browser language, defaults to Arabic
- 🏆 **WC2026 Design** - Premium gold/black/white theme inspired by FIFA World Cup 2026
- 🎯 **High Conversion** - Hero, offers, testimonials, scarcity timer, WhatsApp integration
- 📱 **Responsive** - Mobile-first with Tailwind CSS and glassmorphism
- 🚀 **SEO Optimized** - SSR, JSON-LD, sitemap, robots.txt, Open Graph, Core Web Vitals
- 💬 **WhatsApp CTA** - Floating button + sticky mobile bar + pre-filled message
- ⏰ **Scarcity Timer** - Countdown + limited spots + urgency bar
- 💰 **3 Pricing Tiers** - $9.99/mo, $19.99/3mo, $49.99/yr with savings badges

## 🛠️ Stack

**Next.js 14** · **TypeScript** · **Tailwind CSS 3** · **next-intl 3** · **Lucide React** · **Font Awesome**

## 🚀 Quick Start

```bash
cd iptv-pro
npm install
# Edit .env.local → set your WhatsApp number
npm run dev
```

## 🌐 Default Language

The site defaults to **Arabic (العربية)** and auto-detects the browser language for all other users.

## 🏆 WC2026 Design System

- **Primary Colors**: Black (`#020617`), Gold (`#D4AF37`), White
- **Patterns**: 48-grid geometric pattern (WC2026 inspired)
- **Effects**: Trophy glow, gold shimmer, glass-gold morphism
- **Typography**: Inter (Latin) + Tajawal (Arabic)

## 📊 SEO

- Server-Side Rendering (SSR) ✓

- Open Graph + Twitter Cards ✓

- Robots.txt ✓

## 🚀 Deploy to Vercel

1. Push to GitHub
2. Import to Vercel
3. Set env vars: `NEXT_PUBLIC_WHATSAPP_NUMBER`, `NEXT_PUBLIC_SITE_URL`
4. Deploy 🚀
